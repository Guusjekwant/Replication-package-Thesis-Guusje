# Literature Review Dataset

## Overview

This repository contains the dataset used for a structured literature review on AI adoption.  
The primary data source is an Excel file that synthesizes academic papers, adoption frameworks, determinants, and derived evaluation criteria.

**File included:**
- `Literature review.xlsx`

**Format:** Microsoft Excel (.xlsx)  
**Purpose:** To support transparent, replicable research on AI adoption theories, adoption determinants, and evaluation criteria.

---

## File Structure

The Excel file contains six worksheets (tabs). Each tab represents a distinct stage of the literature review and synthesis process. The tabs are designed to be read sequentially**, from raw literature to ranked criteria.

---

## Tab 1: `Papers`

### Description
This sheet contains the core set of academic papers included in the literature review. Each row corresponds to one paper.

### Purpose
- Defines the literature backbone
- Serves as the reference base for all subsequent analyses
- Enables traceability between theories, determinants, and source studies

### Contents
- Authors
- Publication year
- Title
- Outlet (journal or conference)
- Context or application domain
- Notes on relevance to AI adoption

---

## Tab 2: `Adoption frameworks`

### Description
This sheet lists theoretical models and conceptual frameworks used in the literature to explain AI (or technology) adoption.

### Purpose
- Identifies the theoretical foundations of AI adoption research
- Supports theory-driven comparison and synthesis

### Contents
- Framework or theory name
- Abbreviation (e.g., TAM, TOE, UTAUT)
- Level of analysis (individual, organizational, environmental, or mixed)
- Application areas
- Key authors and references
- Short explanation of each framework
  
---

## Tab 3: `Determinants of AI adoption`

### Description
This sheet contains determinants, factors, and/or constructs identified in the literature as influencing AI adoption.

### Purpose
- Provides an inventory of adoption drivers and barriers
- Forms the empirical and conceptual basis for criteria synthesis

### Contents
- Determinant name
- Definition or explanation
- Level of analysis
- Source papers or authors

---

## Tab 4: `Combined criteria (extended)`

### Description
This sheet presents a comprehensive synthesis of adoption determinants into combined evaluation criteria, with extended explanations.

### Purpose
- Aggregates overlapping or related determinants into higher-level/combined criteria

### Contents
- Combined criterion name
- Extended description
- Underlying determinants
- Associated theories or source references

---

## Tab 5: `Combined criteria (brief)`

### Description
This sheet provides a short version of the combined criteria, focusing on clarity and usability.

### Purpose
- Offers a simplified, practitioner-friendly representation
- Facilitates quick comparison and operationalization

### Contents
- Criterion name
- Short definition or label
- Minimal explanatory text

---

## Tab 6: `Ranked combined criteria`

### Description
This sheet contains the final ranked list of combined AI adoption criteria.

### Purpose
- Establishes relative importance or priority among criteria

### Contents
- Criterion name
- Rank or score
- Weighting or aggregation outcome (if applicable)
